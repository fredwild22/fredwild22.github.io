Can you reach the Director tile?

How to play: Use your arrow keys to move the tiles. When two tiles with the same value touch, they merge into one!

Senior Moderator = 4pts
Server Moderator = 8pts
Junior Staff = 16pts
General Staff = 32pts
Supervisor = 64pts
Shift Manager = 128pts
Ops Manager = 256pts
Senior Ops Manager = 512pts
Deputy Director = 1024pts
